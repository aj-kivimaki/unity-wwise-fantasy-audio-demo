# Footstep Audio System

## Goal

Implement the first gameplay audio system using Wwise and Unity.

## Progress

Created a randomized footstep system using Wwise.

Implemented:

- Footstep Random Container
- Footstep Event
- SoundBank integration
- Animation-driven event triggering

Replaced Unity audio playback with Wwise event posting.

Verified randomized footstep playback during gameplay.

## Lessons Learned

- Wwise Event workflow
- SoundBank management
- Unity Assembly Definition references
- Animation Event driven audio triggering

## Result

The project now contains its first fully functional gameplay audio system implemented through Wwise.
