# Milestone 3 – Original Audio & Portfolio Polish

## Objective

Milestone 3 focuses on transforming the project's technical foundation into a polished portfolio piece.

The architecture is considered complete. This milestone emphasizes original audio content, gameplay feel, final mixing, and presentation.

The goal is to demonstrate not only technical implementation skills but also creative and production-oriented game audio craftsmanship.

---

# Phase 1 – Original Music

## Task 01 – Compose Exploration Theme

Create an original exploration track to replace the temporary music.

Objectives:

- Establish the musical identity of the project.
- Support exploration gameplay.
- Integrate seamlessly into the existing Interactive Music system.

Deliverables:

- Original composition
- Wwise integration
- Documentation
- Gameplay validation

---

## Task 02 – Compose Combat Theme

Create an original combat theme.

Objectives:

# Success Criteria

Milestone 3 is complete when:

- All placeholder music has been replaced.
- Core sound effects are original.
- Gameplay audio feels cohesive.
- Mixing is balanced.
- Documentation is complete.
- The project is suitable for inclusion in a professional game audio portfolio.
